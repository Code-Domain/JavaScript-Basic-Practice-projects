# 🧮 Counter App

A simple and responsive Counter Application built using HTML, CSS, and JavaScript.
This project demonstrates core JavaScript concepts like DOM manipulation, event handling, and conditional logic.

---

## 🚀 Features

* ➕ Increase the counter value
* ➖ Decrease the counter value
* 🔄 Reset the counter to zero
* 🚫 Counter limit protection (±999)
* 📱 Fully responsive design
* 🎨 Clean dark UI with modern card styling

---

## 🛠 Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & Responsive Design
* **JavaScript (ES6)** – Logic & Interactivity

---

## 🧠 How It Works

* The app starts with a default value of `0`.
* When the **Increase** button is clicked, the counter increments.
* When the **Decrease** button is clicked, the counter decrements.
* The **Reset** button sets the counter back to `0`.
* If the counter reaches `999` or `-999`, the app shows an alert and prevents further changes until reset.

JavaScript handles:

* Selecting elements using `querySelector` and `querySelectorAll`
* Adding click event listeners
* Updating the DOM dynamically
* Applying conditional checks for limit control

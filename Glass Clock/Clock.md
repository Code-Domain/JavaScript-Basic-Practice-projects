# 🕒 Glassmorphism Digital Clock

A modern glassmorphism-style digital clock built using **HTML, CSS, and JavaScript**.
It displays real-time hours, minutes, seconds, day, date, month, and year with a clean and responsive UI.

---

## ✨ Features

* ⏰ Real-time clock (updates every second)
* 📅 Displays current day, date, month, and year
* 🎨 Glassmorphism UI design
* 📱 Fully responsive layout
* ⚡ Lightweight and fast (no external libraries)

---

## 🛠 Technologies Used

* HTML5
* CSS3 (Glassmorphism, Flexbox, Media Queries)
* JavaScript (Date object, setInterval)

---

## 📂 Project Structure

```
index.html
```

Everything is written in a single file for simplicity and learning purposes.

---

## 🚀 How It Works

* JavaScript uses the `Date()` object to fetch current time and date.
* `setInterval()` updates the clock every second.
* Arrays are used to map:

  * Day index → Day name
  * Month index → Month name
* The DOM updates dynamically to reflect real-time changes.

---

## 📱 Responsive Design

The layout adjusts for smaller screens using media queries, ensuring the clock looks clean on mobile devices.
